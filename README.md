# amanvaths.github.io

Personal portfolio site for [Aman Vaths](https://github.com/amanvaths) — Founder & CTO of Nadcab Labs.

## Deploy to GitHub Pages

1. Create a new repository named **`amanvaths.github.io`** on GitHub
2. Push this folder to the repo:

```bash
cd amanvaths.github.io
git init
git add .
git commit -m "Launch personal portfolio site"
git branch -M main
git remote add origin https://github.com/amanvaths/amanvaths.github.io.git
git push -u origin main
```

3. Go to **Settings → Pages** and set source to **Deploy from branch → main → / (root)**
4. Your site will be live at **https://amanvaths.github.io/** within 1–2 minutes

## Local preview

Open `index.html` in a browser, or run:

```bash
python3 -m http.server 8080
```

Then visit http://localhost:8080
